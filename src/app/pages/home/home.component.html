Use the NHS Data Dictionary Orchestrator to manage and publish the content of the data dictionary, all via the Mauro Data Mapper platform.

With the NHS Data Dictionary Orchestrator, you will be able to:

  • Review the status of each branch of the NHS Data Dictionary.
  • Carry out integrity checks on each branch and review issues found, which can be rectified in the Mauro Data Mapper.
  • Preview the content that will be published to the final data dictionary.
  • Orchestrate the publication of the data dictionary.

Note that this application is designed for administrators of the NHS Data Dictionary only.
